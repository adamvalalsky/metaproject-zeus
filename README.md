# Resource manager BE


## Description

Resource manager BE.

## Running the app

1. Run bash script `./.build/prepare-dev.sh`
2. Run `docker compose up`

## Test

Unit tests: `./.build/tests-unit.sh` or `npm run test:unit` from `api/` folder
E2E tests: `./.build/test-e2e.sh` or `npm run test:e2e` from `api/` folder
